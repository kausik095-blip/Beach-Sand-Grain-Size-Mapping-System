
### 🌊 Beach Sand Grain Size Mapping System

A low-cost, camera-based automated beach sand grain size mapping system using **ESP32-CAM, Python, OpenCV, and GIS technologies**. The system captures beach sand images, performs pixel-to-millimeter calibration, detects and measures sand grains using image processing techniques, classifies sediment as Fine, Medium, or Coarse sand, and generates statistical reports, histograms, CSV datasets, and GIS-ready outputs for coastal monitoring and erosion assessment.

### 🚀 Key Features

* ESP32-CAM based image acquisition
* Automated grain size measurement
* Pixel-to-mm calibration
* Fine, Medium & Coarse sand classification
* D10, D50, D90 grain size analysis
* Histogram generation
* CSV data export
* GIS-compatible mapping support
* Low-cost and scalable coastal monitoring solution

### 🎯 Applications

* Beach erosion monitoring
* Coastal zone management
* Environmental monitoring
* Climate change impact assessment
* Marine research
* Disaster risk reduction
* Smart coastal infrastructure planning
